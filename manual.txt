---------메뉴바----------
메뉴바에 있는 메뉴는 파일, 도형, 색깔로 나뉜다.

- 파일 메뉴에서는 저장, 열기를 선택할 수 있다.
: 저장 메뉴는 현재 그려져있는 도형을 텍스트 파일에 저장하는 기능이다.
: 열기 메뉴는 텍스트 파일에 있는 데이터를 읽어 도형을 그려주는 기능이다.

- 도형 메뉴에서는 사각형, 타원, 선분을 선택할 수 있다.

- 색깔 메뉴에서는 선 색과 배경색을 선택할 수 있다.
: 선 색은 빨강, 초록, 파랑, 검정 4가지 중에서 선택이 가능하고 기본으로 설정 돼 있는 색은 검정색이다.
: 배경색은 빨강, 초록, 파랑, 하양 4가지 중에서 선택이 가능하고 기보능로 설정 돼 있는 색은 흰색이다.

----------툴바------------
툴바에서는 도형, 선 색, 배경색을 선택할 수 있다.

---------우클릭-----------
기본 : 도형에서 우클릭 되었을 경우엔 해당 도형이 가장 앞으로 나온다. 
만약 빈곳에서 우클릭 되었을 경우엔 그룹이 있었을 경우 그룹을 해제시킨다.
우클릭하고 드래그 되었을 경우 선택된 해당 도형이 앞으로 나오고, 드래그 된 만큼 이동한다.

그룹화 : 우클릭 + 쉬프트 + 드래그 하였을 경우 드래그 된 사각형 영역 안에 들어오는 도형들이 그룹화 된다.
그룹 생성시, 선택된 도형들 중간에 'group'이라고 뜬다. 
그룹을 우클릭 드래그 하면 그룹에 포함된 모든 도형이 드래그 된 만큼 이동된다.

복사 : 우클릭 + 컨트롤 하고 드래그 하였을 경우 드래그 된 사각형 영역 안에 들어오는 도형들이 복사된다.
